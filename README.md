# HR Dashboard UI

```markdown
This HR Dashboard UI is designed to provide a user-friendly interface for human resources professionals to manage various HR-related tasks and access important information efficiently. The UI is built using Bootstrap, a popular front-end framework for developing responsive and mobile-first websites.

## Features

- **Responsive Design:** The UI is fully responsive, ensuring optimal viewing and interaction experience across a wide range of devices and screen sizes.
- **User-Friendly Interface:** The interface is designed with simplicity and ease of use in mind, allowing users to navigate through different sections effortlessly..
- **Analytics and Reporting:** Provides visual analytics and reporting tools to gain insights into HR data, such as turnover trends, recruitment effectiveness, and employee satisfaction.

## Installation

1. Clone the repository:

```bash
git clone https://github.com/your-username/hr-dashboard.git
```

2. Navigate to the project directory:

```bash
cd hr-dashboard
```

3. Open `index.html` in your web browser to view the HR Dashboard UI.

## Usage

- Upon opening the dashboard, users are greeted with an overview of key HR metrics on the dashboard page.
- Navigation links are provided to access different sections of the UI, such as home,action and the details section.
Note that the dashboard page does not contain any links to the settings page , profile and community pages(As this is only the ui of the dashboard)
